# Constants

Shared strings and option arrays live in `src/constants/`.

## Current modules

- `pipelineStages.js`
- `demoStatus.js`
- `roles.js`
- `priorities.js`
- `notifications.js`
- `leadOptions.js`
- `defaultRecords.js`

`src/constants.js` remains as a compatibility layer for older imports. New code should import from `src/constants` or the specific constant module.
