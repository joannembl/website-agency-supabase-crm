# Phase 4 — Notifications

Adds a proper notification center to Agency OS.

## Included

- Bell icon with unread badge
- Right-side notification drawer
- Mark one read / mark all read
- Dashboard notification preview
- Automatic generated notifications for:
  - Follow-ups due today
  - Follow-ups overdue
  - Tasks due today
  - Tasks overdue
  - Demos ready to send
- Saved notifications for:
  - New prospects
  - Tasks created
  - Demo build/status updates

## Supabase

Run `supabase-notifications-migration.sql` after deploying this version.
