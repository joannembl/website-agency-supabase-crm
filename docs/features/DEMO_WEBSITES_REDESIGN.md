# Demo Websites Redesign

This update turns the Demo Websites nav item into a dedicated demo operations dashboard instead of reusing the Pipeline board.

## Added

- Demo Websites page header and stats
- Demo status filter tabs
- Demo project card grid
- Progress timeline per demo
- Demo metadata blocks
- Manage / Build / Activity / Edit actions
- Responsive layout

## Notes

No Supabase migration is required. This page uses existing lead status and demo status inference, and continues to use the existing Demo Manager and Build Demo Website modals.
