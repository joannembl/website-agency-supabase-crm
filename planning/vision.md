# Product Vision

## Mission

Agency OS helps small website agencies run a repeatable demo-first sales and fulfillment process from one place.

## Product promise

Turn a local business prospect into a recurring website client with less manual work.

## Core workflow

Prospect → Research → Build Demo → Send Demo → Follow Up → Proposal → Client → Website Inventory → Revenue → Retention

## Target users

- Solo website agency owners
- Small website agency teams
- Sales reps doing local outreach
- Designers/developers building demo sites
- Clients requesting edits and reviewing work

## Long-term vision

Agency OS becomes the command center for a website agency: CRM, demo builder, proposal generator, client manager, revenue dashboard, website inventory, and automation hub.
