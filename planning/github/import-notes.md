# GitHub Import Notes

This package includes:

- `144` issues in `issues.csv`
- `15` labels in `labels.csv`
- `4` milestones in `milestones.csv`

## Recommended import flow

1. Create labels from `labels.csv`.
2. Create milestones from `milestones.csv`.
3. Import `issues.csv` into GitHub Projects or use GitHub CLI to create issues.
4. Add fields listed in `project-fields.md`.

## Suggested GitHub Project columns

- Inbox
- Backlog
- Ready
- In Progress
- In Review
- Testing
- Done
