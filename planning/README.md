# Agency OS Planning System v1.0

This folder is the single source of truth for the Agency OS product roadmap, backlog, PRDs, architecture notes, and release planning.

Agency OS is a SaaS-style operating system for a demo-first website agency. It supports the workflow from prospecting to demo creation, proposal generation, client conversion, website management, and recurring revenue.

## How to use this planning system

1. Start with `vision.md` and `product-principles.md`.
2. Review `roadmap/roadmap.md` for the release sequence.
3. Use `github/issues.csv` to seed GitHub Issues or a GitHub Project.
4. Use `epics/` and `prd/` before starting major features.
5. Add an ADR in `adr/` whenever a meaningful architecture decision is made.
6. Update `roadmap/backlog.md` after every sprint.
